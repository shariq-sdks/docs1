Service management in Authlete involves configuring and maintaining your OAuth 2.0 and OpenID Connect authorization server.

## Service Configuration

### Basic Settings
- Service name and description
- Issuer identifier
- Service profile selection
- API cluster selection

### Endpoint Configuration
- Authorization endpoint settings
- Token endpoint configuration
- UserInfo endpoint setup
- Discovery endpoint configuration

## Security Settings

### Authentication Methods
- Client authentication methods
- User authentication requirements
- Multi-factor authentication setup
- Session management configuration

### Token Settings
- Access token duration

- ID token settings
- Token signing configuration

### Standard Scopes
- OpenID Connect scopes

- Scope attribute configuration

### Claim Configuration
- Standard claims setup
- Custom claims definition

- UserInfo endpoint claims
