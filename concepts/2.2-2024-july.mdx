# Specs support

## Accept oauth-authz-req+jwt in request object

Updated implementation of `/api/pushed_auth_req` API to accept oauth-authz-req+jwt as value for the typ header parameter in presented request object, as per [JAR/RFC 9101](https://datatracker.ietf.org/doc/rfc9101/).
