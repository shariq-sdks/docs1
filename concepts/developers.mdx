## 概要
FAPI 2.0 とは、セキュリティと相互運用性の向上のために策定された一連の仕様群を指しており、とりわけ金融サービスや銀行業界における利用を主眼としています。FAPI 1.0 から飛躍的な進化を遂げ、多様な分野において高まる課題を解決するために、より高度なセキュリティ対策と洗練された仕様を導入しています。
## FAPI 2.0 Security プロファイル
[FAPI 2.0 Security プロファイル](https://openid.net/specs/fapi-security-profile-2_0-final.html)は、OAuth 2.0 を基盤としたプロファイルであり、様々な業界の API に対して強固なセキュリティ対策を提供することを目的としています。Authlete を使って FAPI 2.0 Security プロファイルを実装するためには、以下の記事を参考にしてください。
- [Authlete を使って FAPI 2.0 Security プロファイルを実装する](/fapi2)
