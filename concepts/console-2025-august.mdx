# Overview

This minor update introduces changes for the Authlete Management Console.
This new version was made available on August 28th (Thu).

# Management Console

## Service principals support

Updated console UI to display audit logs created by service principals.

## JWK Set page load fix

Fixed an issue where the JWK Set page would take longer than usual to load if HSM is not enabled.

## Client import fix

Fixed an issue where services with more than 5 clients would not be imported correctly from Authlete 2.3 to 3.0.

## Other UI improvements

- Fixed minor translation inconsistencies in Japanese
