# Overview of This Release

This is a minor update of Authlete 2.2. It includes the following new or enhanced features since [the version 2.2.22](../2.2.22).

# Newly Supported Standard Specifications

N/A

# New Configuration Items

## Audience Validation (added)

When “Perform” is selected, Authlete checks if an `aud` claim of a request object matches an issuer identifier of a service, and raises an error if they are different.

When “Skip” is selected, Authlete does not do the check as described above.

# Added or Updated APIs

## scopeDetails (added)

`scopeDetails` is a new response parameter of the `/auth/introspection` API. This parameter contains information about scope attributes associated with the target access token.
