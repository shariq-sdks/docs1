# Overview

This hotfix introduces changes for the Authlete Management Console.
This new version was made available on August 12th (Tue).

# Management Console

## Signature algorithms enhancements

Added back some access token signature algorithms which were previously available in Authlete 2.3.

## HSM UI improvements

- Improved HSK UI to mute invalid HSK options and added on-hover tooltips
- Fixed an issue where the HSM switch would not be automatically disabled on the UI when the feature itself is not available on the environment
