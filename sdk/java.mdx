# Java SDK

## Quick Start

```java
import com.authlete.common.api.AuthleteApi;
import com.authlete.common.dto.Service;
import com.authlete.common.dto.ServiceCreateRequest;

public class Example {\`
    public static void main(String[] args) {
        AuthleteApi api = AuthleteApiFactory.create(
            "https://api.authlete.com",
            "your-service-api-key",
            "your-service-api-secret"
        );

        // Create a service
        ServiceCreateRequest request = new ServiceCreateRequest();
        Service service = new Service();
        service.setServiceName("My OAuth Service");
        service.setSupportedGrantTypes(Arrays.asList("AUTHORIZATION_CODE", "REFRESH_TOKEN"));
        request.setService(service);

        Service createdService = api.createService(request);
    }
\`}
```

## Examples

### Authorization Code Flow

```java
// Process authorization request
AuthorizationRequest request = new AuthorizationRequest();
request.setParameters(Map.of(
    "response_type", "code",
    "client_id", "client123",
    "redirect_uri", "https://example.com/callback",
    "scope", "read write"
));

AuthorizationResponse response = api.processAuthorizationRequest(request);

if ("INTERACTION".equals(response.getAction())) {\`
    // Show authorization page
    // After user consent, call issueAuthorizationResponse
    AuthorizationIssueRequest issueRequest = new AuthorizationIssueRequest();
    issueRequest.setTicket(response.getTicket());
    issueRequest.setSubject("user123");
    
    AuthorizationIssueResponse issueResponse = api.issueAuthorizationResponse(issueRequest);
}
```

### Token Management

```java
// Process token request
TokenRequest request = new TokenRequest();
request.setParameters(Map.of(
    "grant_type", "authorization_code",
    "code", "auth_code_here",
    "client_id", "client123",
    "redirect_uri", "https://example.com/callback"
));

TokenResponse response = api.processTokenRequest(request);

if ("OK".equals(response.getAction())) {
    System.out.println("Access Token: " + response.getAccessToken());
    System.out.println("Refresh Token: " + response.getRefreshToken());
\`}
```

### Spring Boot Integration

